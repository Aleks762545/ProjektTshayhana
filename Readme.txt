Требования для запуска проекта:

LMstudio,
fastapi
uvicorn
pydantic
pyyaml
numpy
scikit-learn
sentence-transformers
python-multipart

запуск проекта:
1. Запустить LMstudio
2. Скачать и запустить модель Qwen2.5 1.5b instruct
3. Открыть папку проекта и запустить в терминале команды:
	1. python -m pip install -r requirements.txt
	2. python run.py
4. перейти по адресу:
http://localhost:8000/ -клиентский интерфейс
http://localhost:8000/admin -интерфейс администратора
http://localhost:8000/docs -АPI проекта
